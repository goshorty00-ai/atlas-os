Design a futuristic 3D holographic Smart Home Control Interface for a command centre dashboard.

Overall Style:
Cinematic sci-fi interface inspired by Iron Man JARVIS systems. The UI should feel like a holographic house management system used to control an entire smart home.

Use a dark environment background (#050A12) with glowing neon accents and transparent glass panels. Primary interface colors should be electric blue, cyan, and subtle orange energy highlights.

Main Feature: 3D House Map

Create a central interactive holographic 3D model of a house that appears floating above a circular control platform.

The house should appear semi-transparent with glowing outlines and subtle holographic scan lines.

Rooms should be clearly visible and clickable including:

* Living Room
* Kitchen
* Bedroom
* Office
* Bathroom
* Garage

When hovering or selecting a room, the room should glow and display a floating control panel.

Room Control Panel Features:

* lighting brightness slider
* RGB light color picker
* thermostat temperature control dial
* smart plug on/off toggle
* device status indicators

Controls should look holographic and futuristic with glowing UI rings and animated energy effects.

Sidebar Navigation (Left Side):

* Home
* Media
* AI Assistant
* Smart Home
* Security
* Settings

Icons should glow softly and animate when hovered.

House Status HUD (Top Section):
Display system overview including:

* total lights active
* indoor temperature
* outdoor weather
* security system status
* energy usage

Use futuristic HUD panels with transparent glass and glowing borders.

Scene Automation Controls:
Below the 3D house map add a row of cinematic automation buttons:

Scenes:

* Movie Mode
* Night Mode
* Away Mode
* Party Mode
* Morning Mode

Each scene button should have an animated icon and glowing activation effect.

AI Assistant Panel:
Add a floating assistant module with a holographic orb interface.

Include a command input field:
"Ask the house AI..."

Example commands:

* Turn living room lights blue
* Set house temperature to 21 degrees
* Activate night mode
* Show front door camera

Security Section:
Display camera feeds in holographic panels for:

* Front Door
* Back Garden
* Garage

Visual Design Requirements:

* holographic UI elements
* glassmorphism panels
* neon glow effects
* subtle particle animations
* circular radar style controls
* futuristic typography
* cinematic shadows and lighting

The interface should feel like a high-tech smart home command system from a futuristic control room where the entire house can be managed visually in real time.
