Design an **ultra-futuristic AI-powered Music Library interface** for a cinematic media centre dashboard. This is an advanced **music section add-on**, not a standalone app. The interface must feel like a **high-end AI media command system with holographic visuals and immersive navigation**.

The experience should look like something from a **sci-fi spaceship control deck or Tony Stark's command centre**.

STYLE

• Dark cinematic UI with deep blacks and soft gradient lighting
• Electric blue primary glow with purple and subtle orange accent lighting
• Glassmorphism panels with holographic reflections
• Floating UI elements with depth and parallax
• Ambient particles and subtle animated backgrounds
• Dynamic lighting reacting to music energy

Everything should feel **alive, intelligent, and futuristic**.
