Design a **futuristic AI Media Centre dashboard** for a Windows desktop app module. This UI will be embedded inside a larger program, so **do not create a huge top header, footer, app logo, title bar, or window chrome**. The design should feel like a premium internal module inside an AI command centre.

Create a **dark cinematic sci-fi interface** with glass panels, soft neon cyan/purple accents, subtle gradients, blurred depth layers, premium rounded cards, glowing focus states, and smooth micro-interactions. It should look much more polished than a normal media library, closer to a luxury AI entertainment control room.

## Layout requirements

Use a clean full-screen module layout:

* **Compact collapsible left sidebar**
* Main content area
* No oversized hero header
* No big logo
* No footer
* No wasted empty space
* Everything should fit neatly inside a desktop app viewport
* Design for approximately **1920x1080**, but make it responsive/adaptive
