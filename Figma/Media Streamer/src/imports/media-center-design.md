Design a cinematic desktop media center interface for a streaming application.
The interface will run inside a WPF desktop media center using WebView2, so do not include sidebar navigation, top navigation, or global menus.

The design should only include content pages, because navigation is handled by the host application.

Visual inspiration similar to modern streaming interfaces like Stremio and Netflix.

Target resolution:

1920×1080 desktop UI

Use a dark cinematic theme:

deep black / dark navy background

subtle gradient overlays

neon purple / electric blue accent lighting

glassmorphism panels

smooth shadows

modern sans-serif typography

rounded poster cards

The interface should feel premium, cinematic, and AI-powered.

Create three separate frames/pages.

Page 1 — Home (Shelves Layout)

Create a content discovery dashboard with multiple horizontal media shelves.

Top section:

Large hero banner featuring a highlighted movie.

Include:

movie backdrop
movie title
year
runtime
genre tags
short description

Buttons:

Play
Trailer
Add to Library

Below the banner create horizontal content shelves.

Shelves should include:

Continue Watching
Trending Movies
Popular Movies
Popular Series
AI Recommended For You
Recently Added
Hidden Gems

Each shelf contains poster cards.

Poster card elements:

movie poster
title
rating badge
watch progress bar (if applicable)
hover glow effect

Poster size:

150 × 225 px

Spacing:

16px grid spacing

Shelves should scroll horizontally.

Page 2 — Movie Detail Page

Design a movie information screen.

Layout should be two-column.

Left side = movie information
Right side = streaming sources

Left Section

Top area:

Movie title (large heading)

Below the title show metadata:

Year
Runtime
Age rating
Genre tags

Example:

2024 • 2h 10m • Action • Sci-Fi

Ratings Section

Display ratings as horizontal rating bars.

Sources:

IMDb
Rotten Tomatoes
Metacritic

Each rating should show:

source label
percentage value
colored progress bar

Trailer Section

Include a video trailer preview block.

This should look like an embedded player preview.

Include:

play trailer button
thumbnail backdrop

Movie Description

Add a movie synopsis paragraph.

AI Recommendations

Below the description add a recommended content shelf.

Label:

“Recommended for you”

Poster cards should match the shelf design from the home page.

Right Column — Streaming Sources Panel

Create a floating panel listing available streams.

Each stream row should display:

source provider
video quality badge (720p / 1080p / 4K HDR)
file size
audio language
subtitle indicator
seeder count indicator

Each stream card should include:

Play button
Quality badge
Seeder indicator

Add a stream reliability bar.

Use color indicators:

Green = high seeders
Yellow = medium
Red = low

Highlight one source with:

“Best AI Stream”

Panel style:

glass panel
subtle glow border

Page 3 — Player Page

Create a minimal cinematic video player interface.

Full-screen layout.

Include a video player container centered on screen.

Player controls should include:

Play / Pause
Seek timeline
Preview thumbnails
Volume control
Subtitles selector
Audio track selector
Playback speed
Fullscreen toggle

Extra controls:

Skip intro button
Next episode countdown
Subtitle sync

Controls should appear as floating translucent overlays.

Progress bar should use neon accent color.
